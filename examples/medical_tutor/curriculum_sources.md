## How to Ingest

```bash
# 1. Create curriculum directory structure
mkdir -p curriculum/{m1_cells_tissues,m1_cardiovascular,m1_cns,m1_endocrine}
mkdir -p curriculum/{m1_gi_liver,m1_immunology,m1_infectious_disease}
mkdir -p curriculum/{m1_musculoskeletal,m1_renal,m1_human_growth}
mkdir -p curriculum/{m2_cardiovascular,m2_dermatology,m2_endocrine}
mkdir -p curriculum/{m2_hematology,m2_musculoskeletal,m2_neurosciences,m2_renal}
mkdir -p curriculum/{nursing_fundamentals,nursing_pharmacology}
mkdir -p curriculum/{nursing_skills,nursing_mental_health,health_promotion}
mkdir -p curriculum/histology

# 2. Download materials from the links above into the appropriate directories

# 3. Index each subject with knowledge graph extraction
python examples/medical_tutor/ingest_curriculum.py \
    --docs-path ./curriculum/m1_cells_tissues/ \
    --subject cells_and_tissues --build-kg

python examples/medical_tutor/ingest_curriculum.py \
    --docs-path ./curriculum/nursing_fundamentals/ \
    --subject nursing_fundamentals --build-kg

# 4. Launch the tutor
python examples/medical_tutor/tutor.py
```
